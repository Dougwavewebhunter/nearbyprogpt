NEARBYPRO DEPLOY STEPS

1) In Supabase SQL Editor, run ONLY the newest migration file if you already ran all previous migrations:
   supabase/migrations/20260507143000_nearbypro_request_chat_subscriptions_ui_fix.sql

   Open that file, copy ALL code, paste into Supabase SQL Editor, then RUN.

2) Upload this full extracted project to the ROOT of your GitHub repo.
   Do NOT upload node_modules.

3) Netlify settings:
   Build command: npm run build
   Publish directory: dist
   Base directory: leave blank

4) Netlify Environment Variables must exist:
   VITE_SUPABASE_URL
   VITE_SUPABASE_PUBLISHABLE_KEY
   VITE_SUPABASE_PROJECT_ID

5) Trigger: Clear cache and deploy site.
